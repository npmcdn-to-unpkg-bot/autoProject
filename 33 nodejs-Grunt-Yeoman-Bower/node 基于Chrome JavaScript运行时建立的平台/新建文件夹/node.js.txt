node.js 是一个允许开发人员使用 JavaScript 语言编写服务器端代码的框架。


Node 是一个服务器端 JavaScript 解释器


Node 本身运行Google V8 JavaScript引擎，所以速度和性能非常好，看chrome就知道，

而且Node对其封装的同时还改进了其处理二进制数据的能力。

因此，Node不仅仅简单的使用了V8，还对其进行了优化，使其在各种环境下更加给力。
